    二、執行團隊¶

    1. 課程組¶

    任務： - 開設跨領域課程 - 整合7個系所中心資源 - 統籌11位教師協作

    負責課程： - 國際專業服務領導 - 專業服務學習 - 跨文化溝通實務

    工作內容： - 課程設計與規劃 - 師資協調與安排 - 教學品質控管 - 學生學習評估

    2. 服務組¶

    任務： - 服務場域開發與維護 - 志工服務協調 - 服務品質管控

    服務場域： - 高雄榮民總醫院 - 義大醫院 - 小港醫院 - 國際服務（柬埔寨等）

    工作內容： - 場域關係維護 - 服務排班協調 - 現場問題處理 - 服務時數統計

    3. 研發組¶

    任務： - 數位教材開發 - 教學創新研究 - 服務模式優化

    產出： - 動畫/繪本教材 - 英文教材 - 華文教材 - 線上教學平台

    工作內容： - 教材設計開發 - 教學工具研發 - 成效評估研究 - 創新方案規劃

    4. 培訓組¶

    任務： - 志工培訓規劃 - 專業能力養成 - 持續教育訓練

    培訓內容： - 醫療專業術語 - 翻譯技巧訓練 - 跨文化溝通 - 服務倫理 - 緊急應變

    工作內容： - 培訓課程設計 - 講師邀請協調 - 培訓執行 - 能力認證

    5. 行政組¶

    任務： - 計畫行政支援 - 文書資料管理 - 經費核銷

    工作內容： - 會議安排紀錄 - 公文簽核處理 - 活動行政支援 - 檔案資料管理

    6. 資源組¶

    任務： - 合作夥伴開發 - 資源募集整合 - 對外公關宣傳

    工作內容： - 合作關係建立 - 資源媒合協調 - 媒體公關處理 - 社群媒體經營

    三、教師社群¶

    1132教師社群¶

    成員組成： - 7個系所中心 - 11位跨領域教師

    運作方式： - 定期月例會 - 學期研討會 - 工作坊交流 - 共備共享

    功能： - 課程協作開發 - 教學經驗分享 - 學生輔導支持 - 服務品質把關

    六、運作機制¶

    會議制度¶

    1. 月例會 - 時間：每月一次 - 參與：主持團隊、執行團隊 - 內容：進度報告、問題討論、決策協調

    2. 教師社群會議 - 時間：每月一次 - 參與：社群教師 - 內容：教學分享、課程協作、學生輔導

    3. 志工大會 - 時間：每學期兩次 - 參與：全體志工 - 內容：培訓、分享、聯誼

    4. 合作夥伴會議 - 時間：每學期一次 - 參與：合作機構代表 - 內容：檢討改進、需求溝通、合作深化

    溝通管道¶

    內部溝通： - LINE群組（即時溝通） - Email（正式通知） - 雲端共用文件（資料共享） - 定期會議（面對面交流）

    外部溝通： - Facebook 粉絲專頁 - Instagram 官方帳號 - YouTube 頻道 - 官方網站

    七、品質管控¶

    服務品質管理¶

    1. 行前準備 - 確認服務需求 - 志工能力評估 - 場域環境熟悉

    2. 現場督導 - 教師隨行督導 - 即時問題處理 - 經驗累積記錄

    3. 事後檢討 - 服務經驗分享 - 問題討論改進 - 優良案例表揚

    持續改進機制¶

    PDCA循環： - Plan（計畫）：需求評估、目標設定 - Do（執行）：培訓實施、服務執行 - Check（檢核）：成效評估、問題發現 - Act（行動）：改進措施、標準修訂

    八、成果評估¶

    量化指標¶

    • 服務人次
    • 服務時數
    • 參與志工人數
    • 合作機構數量
    • 教材產出數量

    質化指標¶

    • 服務對象滿意度
    • 志工成長反思
    • 合作夥伴評價
    • 社會影響力
    • 永續發展潛力

## 課程資訊

### 課程內容

文藻外語大學 USR 計畫開設多元化的服務學習課程，包括：

- **國際醫療志工培訓**
- **多語翻譯服務**
- **跨文化溝通**
- **國際服務學習**
- **專業服務領導**

### 服務場域

- 高雄榮民總醫院
- 義大醫院

- 國際服務實踐（暑期）

A: 歡迎所有對國際服務、醫療志工有興趣的文藻學生報名參加。

### Q: 課程時間如何安排？

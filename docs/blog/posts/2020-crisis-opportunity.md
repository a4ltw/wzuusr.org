# 危機就是轉機！

疫情期間，志工服務全面停止，但團隊化危機為轉機，開創新的服務模式。

<!-- more -->

## 疫情衝擊

2020 年新冠肺炎疫情爆發，醫院志工服務全面暫停。對於長期深耕醫院服務的我們來說，這是一個巨大的挑戰。

## 轉型契機

然而，我們選擇將危機視為轉機：

1. **線上服務模式**：發展遠距口譯服務
2. **數位教材開發**：製作線上衛教繪本
3. **虛擬志工培訓**：透過視訊會議進行志工訓練
4. **國際連結**：與柬埔寨建立線上教學連結

## 新的可能

這段期間的嘗試，讓我們發現科技可以突破地理限制，為服務創造更多可能性。

## 省思

> 「每一次的困難，都是成長的機會。」

感謝團隊成員的積極應變，讓我們在逆境中找到前進的方向。

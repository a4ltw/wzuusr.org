# 執行架構 Framework

## 計畫執行架構概述

文藻外語大學 USR 計畫「溫暖白色巨塔的小螺絲釘」以大學社會責任為核心，整合校內外資源，透過跨領域合作推動社會實踐。

---

## 核心團隊編成

### 主持人
負責計畫整體規劃、資源整合與對外聯繫，確保計畫目標達成。

### 共同主持人
協助主持人推動計畫執行，負責特定領域的專業指導與品質把關。

### 協同主持人
支援各項子計畫執行，協調跨單位合作事宜。

---

## 四大主軸

### 1. 國際醫療志工服務
結合外語專業，提供醫療場域口譯服務，協助外籍人士、新住民及移工就醫無障礙。

### 2. 國際教育服務
透過線上英文陪伴課程，服務偏鄉學童及海外弱勢孩童，提升教育素養。

### 3. 永續發展教育
推動 AI 與永續發展主題營隊，培養學生跨域能力與社會關懷意識。

### 4. 數位教材開發
製作雙語繪本動畫、英文教材等數位資源，擴大教育影響力。

---

## 七大執行策略

| 策略 | 說明 |
|------|------|
| **跨域合作** | 整合校內 7 個系所中心、10+ 位教師共同開課 |
| **場域實踐** | 進駐義大醫院、高榮、小港醫院等醫療場域 |
| **國際連結** | 與柬埔寨、馬來西亞等地建立服務學習夥伴關係 |
| **在地深耕** | 服務高雄蚵寮地區國中小學童 |
| **數位創新** | 運用 AI 工具融入課程與服務 |
| **永續經營** | 建立志工培訓系統，確保服務品質與傳承 |
| **成果擴散** | 透過媒體報導、競賽獲獎擴大社會影響力 |

---

## 合作夥伴

- **醫療機構**：義大醫院、高雄榮民總醫院、小港醫院
- **教育單位**：蚵寮國小、蚵寮國中、馬來西亞沙勞越學校
- **國際組織**：柬埔寨服務據點

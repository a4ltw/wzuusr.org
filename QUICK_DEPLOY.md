# 🚀 快速部署指南

您的網站已經構建完成！只需完成以下 3 個簡單步驟即可上線：

## ✅ 第 1 步：重命名分支（必須）

1. 訪問：https://github.com/a4ltw/wzuusr.org/branches

2. 找到分支：`claude/gh-pages-019BaRo8qXiwuHys2PJN57Mj`

3. 點擊分支旁邊的選項按鈕（三個點 ⋮）

4. 選擇「Rename branch」

5. 將分支名改為：`gh-pages`

6. 點擊「Rename branch」確認

## ✅ 第 2 步：啟用 GitHub Pages（必須）

1. 訪問：https://github.com/a4ltw/wzuusr.org/settings/pages

2. 在「Source」下：
   - Branch: 選擇 `gh-pages`
   - Folder: 選擇 `/ (root)`

3. 點擊「Save」按鈕

## ✅ 第 3 步：等待部署（1-3 分鐘）

GitHub 會自動部署您的網站。

部署完成後，訪問：
- **主網站（Wix）**: https://www.wzuusr.org/
- **新網站（MkDocs）**: https://www.wzuusr.org/n/ ⭐
